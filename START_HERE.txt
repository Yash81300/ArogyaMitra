QUICK START
-----------
1. Run setup:
   Linux/Mac: bash SETUP.sh
   Windows:   SETUP.bat

2. Add API keys to backend/.env:
   - GROQ_API_KEY      ‚Üí https://console.groq.com/keys
   - YOUTUBE_API_KEY   ‚Üí https://console.cloud.google.com/apis/credentials
   - SPOONACULAR_KEY   ‚Üí https://spoonacular.com/food-api/console#Profile
   - GOOGLE_CALENDAR_* ‚Üí https://console.cloud.google.com/apis/credentials

3. Start backend:
   cd backend
   source venv/bin/activate  (or venv\Scripts\activate on Windows)
   python main.py

4. Start frontend (new terminal):
   cd frontend
   npm run dev

5. Open http://localhost:3001

URLS
----
Frontend:  http://localhost:3001
Backend:   http://localhost:8000
API Docs:  http://localhost:8000/docs (Swagger UI)
